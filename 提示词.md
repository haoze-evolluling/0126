**项目名称：** 现代杂志风格 · 中国主流网站网络诊断工具

**一、 视觉与交互规范 (Magazine Aesthetics)**

* **配色方案：** 采用复古纸张色（#F5F2E9）背景，文字采用深灰（#1A1A1A）。
* **排版逻辑：** 严格遵循【杂志化布局】。使用超大加粗数字（Display Type）展示核心毫秒数，辅以极小字号的英文标注作为视觉平衡。
* **数据可视化：** 采用【简洁勾线风格】。使用 SVG/Canvas 绘制细线波动图，杜绝阴影与复杂渐变，仅使用单色透明度（如：Green-600/20%）标注状态。
* **状态分级：** 优秀（翠绿）、良好（湛蓝）、一般（琥珀）、极差（朱红），各色独立呈现，保持画面高级感。

**二、 核心技术栈**

* **引擎：** 纯 JavaScript + Fetch API + `performance.now()` 高精度计时。
* **图形：** 轻量级 Chart.js 或原生 Canvas 绘制线性趋势图。
* **图标：** 引入 Lucide-React 或 Font Awesome 矢量图标。
* **响应式：** 适配 Desktop 与 Mobile 的栅格系统。

**三、 功能矩阵**

1. **目标配置：** 预设 50 个中国主流站点（涵盖 BAT、云服务、视频流媒体等），支持一键多选/反选。
2. **诊断引擎：**
* 连续 3-5 次采样，计算平均延时、Jitter（抖动）、丢包分析。
* 实时捕捉 HTTP/HTTPS 握手耗时。


3. **多维展示：**
* **主看板：** 巨型数字显示平均延时，下方配套勾线实时波形。
* **对比墙：** 简洁柱状图（Outline Style）横向对比各站性能。
* **质量仪表盘：** 极简环形线条展示整体网络健康度。


4. **交互控制：** 异步批处理测试（不阻塞 UI），支持结果导出 CSV 与一键快照复制。

**四、 输出要求**

* 生成完整的单文件 HTML（含 CSS/JS）。
* 确保中文语境排版优雅，字间距与行高符合排版美学。
* 逻辑严谨，处理好 Fetch 跨域失败（CORS）时的异常捕获（以连接耗时替代成功响应）。
